## üåü Features

-   **Neon Aesthetic**: Deep black background (`#050505`) with vibrant neon red (`#ff003c`) accents and glowing text effects.
-   **"Buttery" Smooth Animations**: Scroll-triggered fade-in reveals (`IntersectionObserver`), hover lift effects on cards, and bouncy button transitions.
-   **Glassmorphism**: Translucent headers and cards using `backdrop-filter` for a modern, sleek look.
-   **Fully Responsive**: optimized for desktops, tablets, and mobile devices using CSS Grid and Flexbox.
-   **Interactive Elements**:
    -   Glitch/Typing text effects in the Hero section.
    -   Glowing navigation links.
    -   Dynamic social media icons with distinct hover states.

## üìÇ Sections

1.  **Home**: Hero section with a bold introduction and "Glitch" name effect.
